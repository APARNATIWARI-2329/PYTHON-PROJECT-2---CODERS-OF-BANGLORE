# 📊 Coders of Bangalore – Data Analysis Project

Welcome to the **Coders of Bangalore** repository 🚀

This project focuses on collecting, processing, and analyzing social media profile data using Python. The objective is to extract meaningful insights from raw data through parsing, data structuring, and analytical techniques.

---

## 🎯 Project Objectives

* Collect and process profile data
* Parse unstructured text into structured formats
* Analyze social media statistics
* Generate insights from datasets
* Practice real-world data analysis workflows

---

## 🧠 Key Analysis Performed

✅ Profile Data Collection

✅ Data Parsing & Cleaning

✅ Finding Users with:

* Maximum Posts
* Maximum Followers
* Maximum Following

✅ Category Analysis

✅ Structured Data Processing

✅ Statistical Insight Generation
